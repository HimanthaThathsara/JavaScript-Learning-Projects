##  Overview

<code>This repository is a collection of small JavaScript, HTML and CSS learning projects and demos created to practice and demonstrate basic web development concepts. Each subfolder is a standalone project you can open in a browser to see the demo and review the source code.</code>

---

##  Features

<code>
- A variety of beginner-to-intermediate learning projects (canvas drawings, simple apps, demos of browser APIs).
- Each project is self-contained: HTML, CSS and JS files are in the project folder.
- Quick-to-run examples suitable for learning, sharing, and extending.
</code>

##  Getting Started

###  Installation

Install JavaScript-Learning-Projects using one of the following methods:

**Build from source:**

1. Clone the JavaScript-Learning-Projects repository:
```sh
❯ git clone https://github.com/HimanthaThathsara/JavaScript-Learning-Projects
```

2. Navigate to the project directory:
```sh
❯ cd JavaScript-Learning-Projects
```
